# 느낀점
## KNN 회귀를 벗어나며
지난주에 KNN에 대해서 배울 때 KNN은 비모수적 지도학습 "분류기"라고 알았는데, 회귀에도 사용된다는 것을 알고는 어떻게 회귀에 사용될 수 있나 궁금했다. 하지만 의외로 KNN 알고리즘도 회귀에 사용될 수 있었지만, 실제로는 알고리즘의 근본적인 한계 때문에 최대값 / 최소값을 벗어난 이상치에 대해서는 문제가 발생할 수 있음을 알았다.

## 선형회귀 / 다항회귀 / 다중회귀
선형회귀 / 다항회귀 / 다중회귀를 배워보고 나니, 어떻게 다음 값을 계산할지 궁금했다. 가장 먼저 생각한 것은 아무 방정식의 계수에 아무 값이나 넣은 다음에, 오차를 줄여나가는 방식으로 계산하지 않을까했는데.. (경사하강법이라는 명칭이 따로 있었다) 이 방식이 다중회귀에서는 너무도 비효율적이기 때문에 정규방정식을 사용한다고한다. AI 정규스터디에서는 다루지 않았지만 이 이상부터도 새로 알아보고 싶다.

사이킷런에서 회귀 알고리즘도 정규방정식을 사용한다고한다. 그런데 정규방정식은 내부적으로 역행렬을 구해야해서 특성이 많아지면 느려진다는 단점이 있는데, (시간복잡도 O^2.4 ~ O^3) 이를 막기 위해서 SVD / QR 분해를 내부적으로 사용한다고한다. SVD / QR 분해는 작년에 선형대수학 과목에서 PLU 분해를 다루면서 한 번 간략하게 다뤘던 내용이었는데, AI를 공부하면서 선형대수학, 수치해석, 확률과 통계 이런 과목들의 내용이 한 번씩 나오는 것을 보고, 지금까지 내가 배웠던 컴퓨터와 관련된 수학들이 AI에서 다시 모이는게 아닐까 생각들었다. 수학 공부 좀 다시 해야겠다는 생각도 들었다.

SVD / QR 분해 말고도 스터디 때 배웠던 Ridge(L2), Lasso(L1) 규제 역시 대안이 될 수 있다고한다. 모델 학습할 때 복잡한 정도와 단순한 정도를 잘 골라서 학습을 시켜야한다니.. 복잡도를 결정하는게 꽤 어려울 것 같았다.
